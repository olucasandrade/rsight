### Open action decisions (from CONTEXT.md)

**Claude Code:** Open a new Terminal.app window running `claude --resume <conversation_id>`.
- Command: `osascript -e 'tell application "Terminal" to do script "claude --resume <id>"'`
- If claude not found: show status "claude CLI not found — install at claude.ai/code"
- Detect: `which claude` returns exit code 0 if found

**Cursor:** Run `cursor <workspace_path>` where workspace_path is the path component of the conversation's `path` field.
- The `path` field in AiConversation is the full path to `store.db`; workspace is not directly stored.
- Use the conversation `path` parent directory as context, but Cursor is opened without a workspace arg — just `cursor` with no argument (opens Cursor app itself). Actually: the CONTEXT.md says "cursor <workspace_path>" — use the path stored in the result. For Cursor conversations, store the workspace path in the `path` field of AiConversation during Plan 01 cursor.rs implementation, OR derive it from meta (the meta JSON has a `workspacePath` or similar). Since that data may not be available, fall back to opening Cursor app via `open -a Cursor` if no workspace path is determinable.
- If cursor CLI not found: show status error.

**rsight stays open** — use `Command::new(...).spawn()` (not `.output()` which waits).

    ```rust
    /// Open a conversation in its native application.
    /// For Claude Code: opens a new Terminal.app window running `claude --resume <id>`.
    /// For Cursor: opens Cursor via CLI.
    /// rsight stays alive — uses spawn (non-blocking).
    /// Sets status_message on error (CLI not found).
    pub fn open_conversation(result: &SearchResult, status_message: &mut Option<String>) {
        if let SearchResult::AiConversation { conversation_id, path, source, .. } = result {
            match source {
                AiSource::ClaudeCode => {
                    // Check if claude CLI is available
                    let claude_check = Command::new("which").arg("claude").output();
                    if claude_check.map(|o| o.status.success()).unwrap_or(false) {
                        // Open new Terminal.app window with claude --resume <id>
                        let script = format!(
                            "tell application \"Terminal\" to do script \"claude --resume {}\"",
                            conversation_id
                        );
                        let _ = Command::new("osascript")
                            .arg("-e")
                            .arg(&script)
                            .spawn();
                    } else {
                        *status_message = Some(
                            "claude CLI not found — install at claude.ai/code".to_string()
                        );
                    }
                }
                AiSource::Cursor => {
                    // Check if cursor CLI is available
                    let cursor_check = Command::new("which").arg("cursor").output();
                    if cursor_check.map(|o| o.status.success()).unwrap_or(false) {
                        let _ = Command::new("cursor").arg(path).spawn();
                    } else {
                        // Fall back to opening Cursor.app directly
                        let app_check = Command::new("open").args(["-a", "Cursor"]).output();
                        if app_check.map(|o| o.status.success()).unwrap_or(false) {
                            // opened successfully
                        } else {
                            *status_message = Some(
                                "cursor not found — install Cursor from cursor.sh".to_string()
                            );
                        }
                    }
                }
            }
        }
    }
    ```
