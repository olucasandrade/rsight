### Conversation format notes (discovered via codebase inspection)

**Claude Code** — `~/.claude/projects/<project_dir>/<uuid>.jsonl`
- Each line is a JSON object with a `type` field
- `type == "user"` and `isMeta == false` entries are human messages
- First human message where `message.content` (string) does NOT start with `<command` or `<local-command` is the title
- `timestamp` field on the entry = ISO 8601 string (use for date)
- `sessionId` field = conversation ID for `claude --resume`
- Fallback title if no real message found: the JSONL filename stem (the UUID)

**Cursor** — `~/.cursor/chats/<chat_id>/<agent_id>/store.db` (SQLite)
- `meta` table: columns `key` (TEXT) and `value` (TEXT, hex-encoded JSON)
- Hex-decode `value` → JSON: `{"name": "...", "agentId": "...", "createdAt": <ms epoch>, ...}`
- `name` field = conversation title
- `agentId` field = ID for opening with cursor CLI
- `createdAt` milliseconds epoch = date
- Query matching: check if query substring appears (case-insensitive) in `name` OR in any blob `data` (decode as UTF-8, skip non-UTF-8 blobs)
- `~/.cursor/` may not exist — silently skip if absent
- Dependency: `rusqlite` crate must be added to Cargo.toml

### Matching logic
- Claude Code: search the full JSONL content (all message texts) for the query string (case-insensitive substring). If any message matches, emit one `AiConversation` result per conversation file (not per matching message).
- Cursor: search `name` + all blob `data` fields for query (case-insensitive substring). Emit one result per matching agent conversation.
- Cap: both parsers stop sending when the channel is closed (check `blocking_send` errors).

    fn format_cursor_date(ms: u64) -> String {
        // Very simple: just show the approximate month/day
        // Use a basic Julian day calculation
        let secs = ms / 1000;
        let days = secs / 86400;
        // Days since Unix epoch (1970-01-01)
        // Simplified: compute year/month/day via Gregorian calendar algorithm
        let z = days as i64 + 719468;
        let era = if z >= 0 { z } else { z - 146096 } / 146097;
        let doe = z - era * 146097;
        let yoe = (doe - doe/1460 + doe/36524 - doe/146096) / 365;
        let y = yoe + era * 400;
        let doy = doe - (365*yoe + yoe/4 - yoe/100);
        let mp = (5*doy + 2)/153;
        let d = doy - (153*mp+2)/5 + 1;
        let m = if mp < 10 { mp + 3 } else { mp - 9 };

        let month_str = months.get((m as usize).saturating_sub(1)).copied().unwrap_or("???");
